# TRACE Project Context

TRACE is a React + Vite + TypeScript + Electron architectural tracing application.

It is being built as a design tool for importing drawings/images, tracing over them, defining generated areas, and eventually sending those areas to an AI image-generation system.

The core design idea is layered tracing paper:

- Base image layer: imported architectural image/drawing.
- Orange control layer: editable nodes, lines, paths and boundaries.
- Pink generated layer: soft bubble/blob areas generated from the orange controls.
- UI layer: tools, settings, export, generation controls.

The user wants TRACE to feel like architectural software, not a generic image editor.

Important visual language:
- Clean, minimal interface.
- LIMIT DESIGN / TRACE branding.
- Ferrari red accent: #FF2800.
- Dark mode and light mode.
- Keep drawing canvas clear and usable.

The user is the designer/art director. Do not make major design or geometry decisions without asking.
